👋 About Me  
Ms. Magik is an interactive browser-based experience inspired by the classic Magic 8 Ball.  

Designed with a playful tone and personality, the app allows users to ask questions and receive randomized responses in real time. The focus of this project is combining simple logic with engaging UI design to create something fun, responsive, and visually cohesive.

---

✨ Features  

- 🔮 Randomized fortune responses  
- ⚡ Real-time interaction using JavaScript  
- 🎨 Styled, themed interface with personality  
- 🖱️ Responsive user input handling  

---

🛠️ Tech Stack  

- HTML  
- CSS  
- JavaScript  

---

💡 Purpose  
This project was built to reinforce core frontend development skills through a lightweight, interactive application. It demonstrates:

- Dynamic DOM updates  
- Event handling and user interaction  
- Structuring logic for randomized outcomes  
- Building small projects with a clear user experience focus
